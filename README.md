Репозиторий с материалами для воркшопа по Codex / Claude Code и AI-агентам для селлеров маркетплейсов.

Главная идея курса: не просто показать, как AI может проанализировать один отчет, а научить селлера собирать собственный инструментарий:

```text
источники данных -> сбор -> хранилище -> нормализация -> системная память -> AI-проверка -> отчет / действие -> следующий цикл
```

## Единая рамка курса

Основная версия курса - инструментальная. Мы не продаем разовый разбор отчета и не обещаем "полную автоматизацию кабинета". Мы учим селлера собирать систему:

1. Найти источники данных.
2. Получить данные через API, выгрузки, Google Sheets или ручные файлы.
3. Сохранить сырые и обработанные данные.
4. Создать память бизнеса: товары, оферты, поставки, правила маржи.
5. Нормализовать данные в Markdown / JSON / таблицы.
6. Запустить регулярные AI-проверки.
7. Описать скилы, MCP-инструменты, плагины и интеграции.
8. Отдавать результат в отчеты, Google Sheets, Telegram или задачи команде.

## Что должен уметь участник после курса

- Понимать, какие источники данных нужны селлерскому бизнесу.
- Получать данные через API, выгрузки, CSV, Excel или Google Sheets.
- Хранить данные в понятной структуре проекта.
- Создавать системную память бизнеса: товары, оферты, поставки, правила маржи, ограничения.
- Переводить сырые данные в Markdown / JSON / обработанные таблицы.
- Создавать повторяемые AI-проверки.
- Писать скилы и инструкции для регулярных задач.
- Проектировать интеграции через локальные скрипты, MCP, плагины, Google Sheets и Telegram.
- Работать с карточками товаров, изображениями, описаниями и отзывами.
